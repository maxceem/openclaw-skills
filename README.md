# OpenClaw Skills

Some custom or improved original skills for OpenClaw that I use.

- `gog`: improved replacement for the standard `gog` skill, adapted with `SecretRef` support via `GOG_KEYRING_PASSWORD` and improved Google Tasks guidance.
- `groq-whisper-api`: transcribe audio via Groq Automatic Speech Recognition (ASR) models (Whisper), ~10x cheaper than built-in OpenAI's `openai-whisper-api`. Published on [https://clawhub.ai/maxceem/groq-whisper-api](https://clawhub.ai/maxceem/groq-whisper-api).
