# openclaw-skills

## Skills

- `gog-new`: original `gog` skill, adapted with `SecretRef` support via `GOG_KEYRING_PASSWORD` and improved Google Tasks guidance.
- `groq-whisper-api`: transcribe audio via Groq Automatic Speech Recognition (ASR) models (Whisper), ~10x cheaper than built-in OpenAI's `openai-whisper-api`.
